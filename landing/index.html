Quick Start

Launch MLflow with integrated package management

$ pip install -e .
# Install the plugin manager package
$ mlflow server --app-name plugin_manager
# Server running at http://localhost:5000
# Plugin manager at http://localhost:5000/plugin-manager/

Technical Architecture

Implementation details and system requirements

Backend Stack

  • Flask Blueprint for MLflow integration
  • SQLAlchemy for package metadata storage
  • Subprocess calls to pip for package operations
  • REST API endpoints for all operations

Frontend Integration

  • Vanilla JavaScript (no framework dependencies)
  • MLflow 3.0+ compatible UI components
  • Responsive design matching MLflow theme
  • Real-time WebSocket updates (optional)

API Endpoints

  • GET /plugin-manager/installed-plugins
  • GET /plugin-manager/available-plugins
  • POST /plugin-manager/install-plugin
  • POST /plugin-manager/uninstall-plugin
  • GET /plugin-manager/check-updates

Requirements

  • Python 3.8+
  • MLflow 2.0+ (optimized for 3.0+)
  • pip package manager
  • SQLite for metadata storage
